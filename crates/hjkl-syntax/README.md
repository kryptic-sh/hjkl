# hjkl-syntax

Renderer-agnostic syntax-highlighting pipeline for the hjkl editor stack: the
`SyntaxLayer` incremental highlighter, viewport-scoped highlight passes
(`build_by_row`), and a renderer-agnostic `RenderOutput`.
