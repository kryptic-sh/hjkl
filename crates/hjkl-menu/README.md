# hjkl-menu

Renderer-agnostic context menu data model for the hjkl editor stack. MenuAction
hierarchy, open/close state, keyboard navigation, accelerator matching, and
dispatch. Rendering lives in hjkl-menu-tui / hjkl-menu-gui.
